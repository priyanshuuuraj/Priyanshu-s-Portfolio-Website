# 🌐 Priyanshu Raj — Developer Portfolio

A modern, responsive personal portfolio website built with React + Vite and Tailwind CSS. Features smooth animations, typing effects, and a working contact form.



## 🛠️ Tech Stack
- ⚛️ React.js + Vite
- 🎨 Tailwind CSS
- ✍️ React Typing Effect
- 📧 EmailJS (contact form)
- ☁️ Vercel (deployment)

## 📁 Features
- Responsive design for all screen sizes 📱💻
- Smooth scroll and animation effects ✨
- Typing animation with multiple roles 🎯
- Working contact form with EmailJS 📬
- Clean and modular React components 🧩
- Professional sections: Hero, About, Skills, Projects, Contact 👨‍💻

## 🧑‍💻 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/priyanshuuuraj/Priyanshu-s-Portfolio-Website.git
cd Priyanshu-s-Portfolio-Website
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
